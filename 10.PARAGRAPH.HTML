HTML elements are the building of HTML pages. With HTML constructs, images and other abject such as interactive forms may embedded into the rendered page.
HTML provides a means to create structured documents by denoting structural semantics for text such as headings,paragraphs,lists,links,quotes and other items.
HTML elements are delineted by tags,Written using angle brackets. Tags such as <img/> and <input/> directly introduce content into the page.
Other tags such as <p> surround and provide information about document text and may include other tags as sub-elements.
Browsers do not display the HTML tags use interpret the content of the page.
